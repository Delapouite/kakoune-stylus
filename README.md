# kakoune-stylus

[kakoune](http://kakoune.org) plugin to handle [stylus](http://stylus-lang.com) files.

## Install

Add `stylus.kak` to your autoload dir: `~/.config/kak/autoload/`

## Usage

This plugin provides highlighting and indent hooks for `*.styl` files.
